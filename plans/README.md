## Findings considered and rejected

- **Graft a discounted terminal reward into GRPO's game_ctx**: no-op — the
  historical episode outcome is constant across a prompt's sample group, and
  GRPO's group-normalized advantage cancels per-prompt constants exactly.
  Cross-turn credit is already carried by the potential-shaped elimination
  reward. Documented in plan 003 (docs/HYBRID_RL.md paragraph).
- **Further gate tuning (margin below 0.01)**: the gate is not the bottleneck
  after plan 001; margins below ~2σ promote noise.
- **More GRPO on the raw 350M at standard difficulty**: measured dead end
  (July 2026: flat 0%, one entropy collapse); revisit only after 002+006.
- **Counting pruner-assisted wins as model skill**: permanently rejected —
  eval-gate purity is a design invariant (D11, HYBRID_RL.md); `eval_pruned`
  stays a recorded-only product metric.
- **Adopting OpenRL/Gymnasium as a dependency**: the Game ABC already mirrors
  the Gymnasium contract (D1); `bridges/gym_adapter.py` remains the planned
  integration point for external envs. Patterns were adopted (vectorization,
  replay), not the frameworks.
